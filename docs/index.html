Platform Components

analytics

Anomaly detection, machine passport builder and REST API.

industrial-exporter

Multi-protocol ingestion for OPC UA and MQTT with semantic mapping.

ui

React interface for machine registry, monitoring and diagnosis.

simulators

Reproducible CNC simulators over OPC UA and MQTT.

Living Machine Passport
for Industrial Assets

Integrates observability and digital passport into a single persistent artefact per machine. Covers identity, connectivity, semantic mappings, monitoring, diagnosis and lifecycle traceability.

Validated with CNC simulators over OPC UA and MQTT in controlled, reproducible scenarios.

TERMINAL

cd Operational-Machine-Passport
docker compose up -d --build

# UI           http://localhost:4000
# Grafana      http://localhost:3000
